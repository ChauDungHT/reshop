# PROMPT 02: TRANG CHỦ & DANH SÁCH SẢN PHẨM (CATALOG)

**Vai trò:** Frontend Developer
**Nhiệm vụ:** Xây dựng giao diện duyệt sản phẩm cho khách hàng.

### Yêu cầu chi tiết:
1.  **Trang chủ (Home):**
    *   Khu vực Banner (Ảnh tĩnh).
    *   Khu vực "Sản phẩm nổi bật": Gọi API với filter `is_featured=true`.
    *   Khu vực "Mới nhất": 8 sản phẩm mới nhất.

2.  **Trang Danh mục (Product Listing Page):**
    *   **Filter Sidebar**: 
        *   Checkbox danh mục.
        *   Slider/Input khoảng giá.
        *   Radio chọn số sao (4*+).
    *   **Grid sản phẩm**: Hiển thị 4 cột trên Desktop, 2 cột trên Mobile.
    *   **Phân trang**: Nút điều hướng trang.

3.  **Layout**: Đảm bảo sử dụng `DashboardLayout` cho các trang dashboard và layout chung cho storefront.

**Output mong đợi:** Giao diện Home và Catalog hoàn thiện, có dữ liệu mẫu hoặc gọi API thật.
